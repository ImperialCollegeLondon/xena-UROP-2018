(PB0105) Let $A$ be the set $\{1,2,3,4,5\}$. Which of the following statements are true and which are false? (just write T or F).
\begin{multicols}{2}
\begin{enumerate}[(a)]
\item $1\in A$. 
\item $\{1\}\in A$.
\item $\{1\}\subseteq A$.
\item $\{1,2\}\subseteq A$.
\item $\{1,2,1\}\subseteq A.$
\item $\{1,1\}\in A.$
\item $A\in A$.
\item $A\supseteq A$.
\end{enumerate}
\end{multicols}
